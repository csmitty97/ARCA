What is HTML?

HTML is a mark-up language for creating web pages. HTML stands for Hyper Text Markup Language. Click here for an introduction to HTML

What is an HTML element?

An element is defined by a start tag, some content, and an end tag. An example is

<p>Welcome To HTML<p>

To learn more about HTML elements click here.

What is an HTML Attribute?

Attributes provide additional information about HTML elements. All elements can have attributes, however they must be specified in the start tag. Click here to learn more about attributes.
